# Kerigma

> Herramienta de preparación homilética para predicadores, pastores y líderes de célula.

**Kerigma** guía al predicador a través de cada etapa del sermón — desde el gancho de apertura hasta el llamado final. Organiza el contexto del pasaje, la exégesis, la proposición homilética, los puntos de desarrollo, las ilustraciones y la aplicación en una plantilla estructurada que se exporta con un clic.

Diseñada para quienes quieren predicar con profundidad, claridad y propósito.

---

## Características

- **Plantilla estructurada** — 8 secciones que cubren todo el arco del sermón: gancho, contexto, exégesis, proposición, puntos principales, ilustraciones, aplicación y conclusión.
- **Barra de progreso** — indica visualmente qué tan completa está la preparación.
- **Exportación en texto plano** — genera un documento listo para imprimir o guardar.
- **Copia al portapapeles** — con un clic, todo el contenido queda disponible para pegar en Word, Notion o cualquier editor.
- **Sin dependencias** — un único archivo HTML, sin frameworks ni librerías externas. Funciona en cualquier navegador.
- **Dark mode** — se adapta automáticamente a la preferencia del sistema operativo.
- **Responsive** — usable en móvil, tablet y escritorio.

## Uso

No requiere instalación. Descarga el archivo y ábrelo en cualquier navegador:

```bash
git clone https://github.com/tuusuario/kerigma.git
cd kerigma
# Abre index.html en tu navegador
open index.html
```

O accede directamente desde el navegador si está publicado en GitHub Pages.

## Secciones de la plantilla

| # | Sección | Propósito |
|---|---------|-----------|
| 1 | Gancho de apertura | Primera impresión — atrapa la atención |
| 2 | Contexto del pasaje | Quién, cuándo, dónde, a quién |
| 3 | Exégesis | ¿Qué dice el texto exactamente? |
| 4 | Proposición homilética | Una oración — el sermón en miniatura |
| 5 | Puntos principales | 2-4 movimientos que desarrollan la proposición |
| 6 | Ilustraciones | La ventana que deja entrar la luz |
| 7 | Aplicación | ¿Qué debe hacer, creer o sentir la audiencia? |
| 8 | Conclusión y llamado | El aterrizaje — no un resumen sino un impulso |
